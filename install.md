# 🧩 Installation Guide

The project includes a fully automated installer: **`install.ps1`**.  
This script sets up the entire environment — Python, dependencies, tools, and models — with no manual steps required.

This document explains how to run the installer and what it does.

---

## ▶️ Running the Installer

From the project root:

```powershell
pwsh ./scripts/install.ps1
